# Image Specifications

All images generated by `seo-image-generate` follow these specifications for SEO and social sharing compatibility.

---

## Image Types and Dimensions

### Featured Image (OG / Social Share)

- **Dimensions:** 1200 × 630 px
- **Format:** WebP
- **Purpose:** Open Graph (`og:image`), Twitter Card, article hero image
- **Naming:** `featured-{slug}.webp` where `slug` is the article slug from frontmatter

This image appears when the article is shared on social media. It must be visually clear at thumbnail size and represent the article's main topic.

### Section Images

- **Dimensions:** 800 × 450 px
- **Format:** WebP
- **Purpose:** Inline article images, one per H2 section
- **Naming:** kebab-case derived from the H2 section heading

**File naming rule for section images:**
1. Take the H2 heading text
2. Lowercase all characters
3. Replace spaces and special characters with hyphens
4. Remove trailing hyphens
5. Append `.webp`

Examples:
- `## Best Running Shoes for Flat Feet` → `best-running-shoes-for-flat-feet.webp`
- `## How to Choose Your First Pair` → `how-to-choose-your-first-pair.webp`
- `## Q&A: Expert Tips` → `qa-expert-tips.webp`

---

## Alt Text Requirements

- **Maximum length:** 125 characters
- **Must include:** primary keyword, used naturally (not stuffed)
- **Must describe:** what is visually depicted in the image
- **Must not:** start with "image of", "photo of", "picture of" (redundant)
- **Must not:** be identical to the image filename or section heading

**Good alt text examples:**
- `Running shoes for beginners displayed on a wooden surface with a track in background`
- `Close-up of cushioned sole on a beginner running shoe`
- `Runner wearing lightweight beginner running shoes on a forest trail`

**Bad alt text examples:**
- `best-running-shoes-for-beginners.webp` (filename as alt)
- `Image of running shoes` (starts with "image of")
- `best running shoes for beginners best running shoes for beginners` (keyword stuffed)

---

## Image Output Structure

All generated images and their metadata are recorded in `image_map.json` using the schema in `references/image-map-schema.json`.

The image map is consumed by `seo-publish-cms` to attach the featured image and inline section images when publishing to WordPress.

---

## WebP Encoding Notes

When calling an image generation API, request WebP format output. If the API returns PNG or JPEG:
1. Accept the image as delivered
2. Note the actual format in `image_map.json` under a `format` field
3. Conversion to WebP can be done post-generation with ImageMagick: `convert input.png -quality 85 output.webp`

---

## Prompt Quality Guidelines

Good image prompts for SEO articles follow this pattern:

```
[Subject description], [context/setting], [style], [lighting], [no text overlays]
```

Examples:
- `A pair of white lightweight running shoes on a wooden surface, outdoor park in background, clean product photography, soft natural lighting, no text`
- `Beginner runner in athletic wear jogging on a forest trail at sunrise, candid action shot, warm golden light, no text overlays`

Avoid:
- Prompts with text instructions to add words/labels to the image
- Abstract or artistic styles (photorealistic works best for SEO content)
- Overly complex scenes with many subjects
